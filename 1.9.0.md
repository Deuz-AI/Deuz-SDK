# v1.9.0 — "Doğrulayan, Öğrenen, Zapt Eden Runtime"

> **Kapsam kararı (2026-07-22):** 1.8.0 "Otonom Ajan Runtime" (workspace, CodeAct, planner→verifier, background runs, browser, `./providers`, `./testing`) yeşil kapıdan geçtikten sonra. Bu belge, 2026 literatür taraması + beyin fırtınasının **kod-doğrulanabilir** release planıdır.
>
> **Kaynaklar:** FineVerify (2606.00660), LLM-as-a-Verifier (2607.05391), SEVA (2606.29713), AgentV-RL (2604.16004), LHTB (2607.08964), Self-GC (2607.00692), SmoothAgent (2607.00151), ACE (2510.04618), All-Mem (2603.19595), NeuroTaint (2604.23374), FIDES (2505.23643), Agent Traces→Trust (2606.04990), Isolation survey (2607.12406), Self-Healing Orchestrator (2606.01416), MAST, SkillOpt / TT-SI parametresiz kol, cascade routing (dekoninck25) + RLM-Cascade (2606.22840), mem0 State of AI Agent Memory 2026.
>
> **Önkoşul:** 1.8.0 yayınlanmış veya en azından `verifyStep`, `./testing` (`runEval`/`createMockModel`), `./runtime`, `Workspace`, `ComputeSandbox`, `planTasks` / `bestOfN` yüzeyleri kilitli.
>
> **Değişmezler:** edge-safe çekirdek (lint), zero runtime dep, kanonik `StreamPart` hattı, immutable history, G1/G2, golden-replay, surface append-only. LLM'li her mekanizma seam üstünden, opt-in, maliyeti `onUsage`+pricing ile görünür.

## Hedef

| Senaryo | 1.8.0 sonrası (öngörü) | 1.9.0 sonrası | Rakip (Mastra / AI SDK 7, öngörü) | Farkın kaynağı |
| --- | --- | --- | --- | --- |
| Chatbot | ~%82 | **~%90** | ~%78–85 | yapısal doğrulama + sahte-bitiş guard + cascade maliyet |
| CLI / otonom ajan | ~%86 | **~%93** | ~%80–86 | skills write-path + held-out gate + MAST diagnostics |
| AGI-track (uzun-ufuk) | ~%72 | **~%84** | ~%65–72 | governContext + memory maintainer + LHTB-stili graded eval |

Konum cümlesi:

> **Üreten değil — doğrulayan, öğrenen ve zapt eden runtime.** Sıfır bağımlılık; her iddianın yanında golden-replay ve hakemli kaynak.

Pazarlama manşeti (hepsi 2026 makale-gerekçeli):

1. **Yapısal doğrulayıcı** — alt-check + kanıt + kalibre güven (FineVerify / SEVA); AI SDK / Mastra'da kütüphane-primitifi yok.
2. **Sahte-bitiş korkuluğu** — LHTB'nin 1 numaralı darboğazı; frontier modeller bile erken duruyor.
3. **Skill yazımı + held-out kapı** — edit ancak `runEval` skorunu düşürmüyorsa kabul (SkillOpt); deterministik gate yalnız bizde.
4. **MAST-aware tanılama** — 14 hata modundan sinyal-tespit edilebilir olanlar loop'ta.
5. **Context nesne yaşam döngüsü** — Self-GC: fold/mask/prune prova→commit; workspace sidecar.
6. **Cascade maliyet kolu** — draft→verify yanıt düzeyinde; tool-seçim turunda bypass (RLM-Cascade dersi).

**2.0'a bilinçli erteleme (bu belgede taslak, 1.9.0'da uygulanmaz):** CaMeL-lite provenance/taint + `ToolPolicy` + lethal-trifecta linter + provenance-güdümlü kurtarma (NeuroTaint/FIDES/Isolation survey). Bunlar kanonik hat + durable üstünde kategorik farktır; ayrı majör.

## API taslakları (additive, kilitlenecek yüzey)

```ts
// N1 — yapısal doğrulayıcı
type VerifierCheck = { id: string; question: string; pass: boolean; evidence?: string };
type VerifierResult = {
  ok: boolean;
  confidence?: number; // 0..1
  feedback?: string;
  errorCategory?: 'hallucination' | 'incomplete' | 'tool-mismatch' | 'format' | 'other';
  checks?: VerifierCheck[];
  evidence?: unknown;
};
function createVerifier(options: {
  model: LanguageModel;
  system?: string;
  maxChecks?: number;
}): {
  verify(input: { goal: string; answer: string; context?: Message[] }): Promise<VerifierResult>;
  score(candidate: string, goal: string): Promise<number>;
};

// N2 — sahte-bitiş
type DoneWhen = (ctx: {
  text: string;
  messages: Message[];
  usage: Usage;
  stepIndex: number;
}) => boolean | Promise<boolean>;
// CommonCallOptions.doneWhen?: DoneWhen
// CommonCallOptions.falseFinishGuard?: boolean | { maxRetries?: number }

// N3 — graded eval
function runGradedEval<I>(
  cases: Array<{ name: string; input: I; subtasks: Array<{ check: (out: unknown) => boolean | Promise<boolean>; weight?: number }> }>,
  run: (input: I) => Promise<unknown>,
): Promise<{ score: number; results: Array<{ name: string; score: number; passed: number; total: number }> }>;

// N4 — skill gate
function proposeSkillEdit(input: {
  name: string;
  from?: string; // existing SKILL.md
  trajectory: Message[];
  model: LanguageModel;
}): Promise<{ diff: SkillEdit[]; markdown: string }>;
function createSkillGate(options: {
  cases: EvalCase<unknown, unknown>[];
  run: (skillMarkdown: string) => (input: unknown) => Promise<unknown>;
}): (edit: { markdown: string }) => Promise<{ accepted: boolean; before: number; after: number; report: EvalReport }>;

// N10 — cascade
function createCascade(
  models: [LanguageModel, LanguageModel],
  options?: {
    accept?: (draft: { text: string; finishReason: FinishReason }) => boolean | Promise<boolean>;
    /** default: true — tool / tool_choice required turns skip draft */
    bypassOnTools?: boolean;
  },
): LanguageModel; // veya middleware
```

## Rubrik / kanıt disiplini

1. Her "rakipte yok" iddiası yayın haftasında Mastra + AI SDK docs'a karşı yeniden doğrulanır; çürüyen satır changeset'ten düşer.
2. N1–N4 için en az bir golden-replay + bir graded-eval senaryosu repo'da.
3. LHTB'ye tam bağlanmak zorunlu değil; **yerel 5–10 görevlik dense-graded set** (coding + tool + false-finish) README'ye sayı ile işlenir.
4. Güvenlik iddiası 1.9.0'da **yok** — "provable" / CaMeL dili yasak; 2.0'da AgentDojo-tarzı ölçüm + kapsam sınırı.

## Riskler

1. **Kapsam şişmesi** — N1–N10 tek minor'a sığmazsa: Sprint 1–2 = **1.9.0 çekirdek**, Sprint 3–5 kesilebilir → **1.9.1+**. Bu belge sürüm adı **1.9.0**.
2. **Verifier maliyeti** — her completion ekstra çağrı. Tedbir: opt-in; `confidence` düşükse skip; cascade ile ucuz model.
3. **Skill gate false negative** — fazla katı gate öğrenmeyi öldürür. Tedbir: eşik konfigüre; rejected-edit buffer insan onayı.
4. **governContext prefix-cache** — kötü commit KV'yi kırar. Tedbir: prova + "yalnız en eski dilim" kuralı (mevcut compaction ile aynı).
5. **Rubrik meşruiyeti** — kendi skorumuz. Tedbir: hakemli kaynak + ham çıktı.
